# LOGLITE

I will try and create a simple, lightweight, and easy to setup (and use) logging system

## Configuration

LogLite allows you to configure its behavior using a YAML configuration file.

## Example `config.yaml`

```yaml
debug_level: 'DEBUG'
protocol: 'HTTP'
port: 8080
log_file: 'logs/app.log'
max_connections: 50
```
